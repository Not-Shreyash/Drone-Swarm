Single-Drone System

Autonomous Drone System

  • An Autonomous Drone System is an unmanned aerial vehicle (UAV) capable of performing tasks with minimal or no human intervention.
  • It integrates hardware, software, communication protocols, and control algorithms into a single cyber-physical system.
  • The system continuously senses its environment, processes information, makes decisions, and executes control commands.
  • Modern autonomous drones are widely used in surveillance, mapping, agriculture, logistics, inspection, search and rescue, and military operations.

Characteristics and Functionalities

  • Characteristics:
    • Autonomous Navigation and sensor based perception
    • Real-time decision making and control
    • Wireless communication and mission execution
  • Functionalities:
    • Environment sensing and localization
    • State estimation and flight control
    • Data logging and path planning

ROS2 (Robot Operating System 2)

  • It is an open-source robotics middleware.
  • ROS2 enables the communication between distributed software components.
  • It provides standardized tools for robotic application development.
  • Its major components are Nodes, Publishers, Subscribers, Parameters, etc.

DDS (Data Distribution Service)

  • DDS is the communication middleware used by ROS2.
  • It removes the need for a central master server.
  • It supports real-time distributed applications
  • Its main features are automatic discovery, configurable QoS, security and scalability.

PX4 Autopilot

  • PX4 is an open-source flight control software for drones and other unmanned vehicles.
  • It provides a complete autopilot solution, including flight control algorithms, sensor drivers, and communication protocols.
  • PX4 supports various vehicle types, including multirotors, fixed-wing aircraft, and VTOLs.
  • It is compatible with various hardware platforms and integrates with ROS2 for advanced functionalities.

ROS2 and PX4 Integration

  • Sensor data are collected and processed by ROS2 nodes.
  • PX4 publishes the topics for communication with ROS2.
  • DDS transports the messages between the nodes.
  • ROS2 receives and processes the data for further analysis and control.
  • New commands are generated and sent back to PX4.

State Estimation and Mission Execution

  • State estimation involves determining the position, velocity, and orientation of the drone.
  • Mission execution refers to the autonomous navigation and task completion capabilities of the drone.

Limitations of Single Drone Systems

  • Limited coverage area and payload capacity.
  • Single point of failure, leading to mission failure if the drone malfunctions.
  • Limited scalability for large-scale operations.
